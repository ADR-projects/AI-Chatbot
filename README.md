## ☕️ ChillChatBot

A cozy conversational companion powered by Gemini 2.5 Flash.  

---

### 🔧 Stack
- Frontend: Vite + React
- Backend: JavaScript
- AI: Gemini 2.5 Flash via API
- Assets: Custom pixel art crafted in Aseprite

---

### 🎨 Interface
- Smooth, simple, and chill UI  
- Designed to keep things lowkey and friendly  
- Just chat and relax.

---

### 🚀 Running the Bot
```bash
npm install
npm run dev
